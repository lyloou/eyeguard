# EyeGuard 护眼卫士 — 产品计划

> 基于产品视角盘点，逐一实现。

---

## P0 — 核心体验补全

- [ ] [P0-1] 系统通知（UNUserNotification）
  工作结束/休息开始时即使息屏也能收到提醒
- [ ] [P0-2] 音效提示（NSSound）
  休息开始、休息结束播放轻提示音
- [ ] [P0-3] App 图标 & 状态栏图标
  AppIcon（1024x1024）+ 状态栏 template image（SF Symbol 短期方案）

---

## P1 — 日常使用完善

- [ ] [P1-1] 登录启动（SMAppService）
  支持开机自启，macOS 13+ 原生 API
- [ ] [P1-2] 全局快捷键
  ⌘⇧E 启动、⌘⇧P 暂停，菜单栏 App 全局响应
- [ ] [P1-3] 深色/浅色模式适配
  状态栏绿色背景、弹窗样式随系统主题变化

---

## P2 — 长期留存功能

- [ ] [P2-1] 国际化（i18n）
  所有 UI 文字提取为 `NSLocalizedString`，支持英文
- [ ] [P2-2] 今日统计
  今日工作轮次、累计休息时长，藏菜单底部
- [ ] [P2-3] 首次启动引导
  3 步新手指引图（开始工作 → 到点休息 → 按键继续）
